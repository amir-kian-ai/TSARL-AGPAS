# Public release policy

This repository is a timestamped public preview of an unpublished research codebase.

## Allowed interpretation

Readers may inspect:
- file names,
- API names,
- result summaries,
- high-level method descriptions, and
- public metadata.

## Not provided in this preview

The following are intentionally omitted:
- complete training code,
- complete evaluation scripts,
- exact algorithmic internals,
- reproducibility-ready experiment recipes,
- full checkpoints,
- complete ablation logs.

## Rationale

The project is under active manuscript preparation and peer-review planning. A fuller release may follow after publication, acceptance, or an explicit author decision.

## Contact

Please contact the repository owner for collaboration, licensing, or pre-publication access requests.
